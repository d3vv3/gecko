# gecko

A self-hosted open-source voice assistant.

## Road map

- [x] Voice Activity Detection (VAD)
- [x] Text-to-Speech (TTS) using whisper
- [x] Perplexity AI responses
- [ ] Speech-to-Text (STT)

## Development

It is known to work on python 3.12.

```bash
pip install -r requirements.txt
```

```bash
cd src
python app.py
```
